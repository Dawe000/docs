This page walks you through everything you need to make your first successful request to the ag0 API. By the end, you'll have an account, a working API key, and a live API response — all in under five minutes.

<Steps>
  <Step title="Create an account">
    Go to the ag0 dashboard at [https://app.ag0.io](https://app.ag0.io) and sign up for a free account. You can register with your email address or authenticate through GitHub or Google. Once you confirm your email, you'll land in the main dashboard.
  </Step>

  <Step title="Get your API key">
    From the dashboard, navigate to **Settings → API Keys**. Click **New Key**, give it a descriptive name (for example, `my-first-key`), and click **Create**. Copy the key immediately — ag0 displays it only once at creation time.

    Your key will look like this:

    ```
    ag0_sk_abc123xyzDEF456uvwGHI789jkl
    ```
  </Step>

  <Step title="Make your first API call">
    Use the key you just copied to call the resources endpoint. Replace `YOUR_API_KEY` with your actual key:

    ```bash
    curl -X GET https://api.ag0.io/v1/resources \
      -H "Authorization: Bearer YOUR_API_KEY" \
      -H "Content-Type: application/json"
    ```

    A successful response looks like this:

    ```json
    {
      "object": "list",
      "data": [
        {
          "id": "res_01hx29kqz4fn8vt3mdcbr7ewys",
          "name": "default-workspace",
          "type": "workspace",
          "created_at": "2024-11-01T10:22:00Z",
          "status": "active"
        }
      ],
      "has_more": false,
      "total_count": 1
    }
    ```

    A `200 OK` with a `data` array means everything is working correctly.
  </Step>

  <Step title="Explore further">
    Now that your first call is working, keep building:

    - Read the [Concepts Overview](/concepts/overview) to understand how ag0 resources, workspaces, and integrations fit together.
    - Follow the [First Integration guide](/guides/first-integration) to connect an external service and trigger your first automated workflow.
  </Step>
</Steps>

<Tip>
  Running into issues? Reach out to the ag0 support team at **support@ag0.io** or open a ticket directly from the dashboard. We typically respond within one business day.
</Tip>
